Web-ViiR

Virus Identification Independent of Reference sequences

A local, one-click virus/viroid detection workflow for plant RNA-Seq datasets. Runs entirely inside Docker with no Linux expertise required.

Web-ViiR v0.9 (local Docker edition)

What This Tool Does

Web-ViiR automatically performs the complete workflow from FASTQ upload to differential expression, viral domain scanning, rRNA filtering, k-mer profiling, and BLASTn annotation. It runs entirely on your local machine (Docker), requires no Linux knowledge, and produces publishable-quality results.

Input Requirements

Provide paired-end RNA-Seq FASTQ files for two groups (N & V). Example filenames:

  • R_DKDL250007304-1A_233VFTLT3_L2_1.fq.gz
  • R_DKDL250007304-1A_233VFTLT3_L2_2.fq.gz

Format: paired-end (xxx_1.fq.gz + xxx_2.fq.gz)

Output Overview

  • Trinity assemblies & expression matrices
  • DESeq2 DEG tables & volcano plots
  • Pfam viral domain hits & rRNA summaries
  • k-mer viral signatures & BLASTn annotations
